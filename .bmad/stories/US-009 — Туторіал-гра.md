# US-009 — Туторіал-гра

## User Story
**Як** дитина,
**я хочу** зіграти коротку вступну гру після вибору аватара,
**щоб** зрозуміти як працюють ігри в додатку (тап, drag-and-drop, зірки, XP).

## Acceptance Criteria

### AC-1: Коли запускається туторіал
- **Given** аватар обрано (US-008), вітальна анімація завершена
- **When** автоматичний перехід
- **Then** відкривається `/(main)/onboarding/tutorial`
- **And** персонаж-компаньйон (відповідний віковій групі) каже: "Давай спробуємо разом!" (озвучка + текст)
- **And** для 🐣 малюків: озвучка ОБОВ'ЯЗКОВА
- **And** для інших груп: озвучка + текст bubble

### AC-2: Layout екрану туторіалу
- **Given** екран туторіалу
- **When** бачу
- **Then** layout:

```
┌────────────────────────────────┐
│                    Пропустити  │  ← text link, right, caption, textSecondary
│                                │
│  [Персонаж-компаньйон]        │  ← 80dp, зліва
│  💬 "Давай спробуємо разом!"  │  ← speech bubble справа від персонажа
│                                │     bubble: surface bg, border-radius xl
│                                │     tail вказує на персонажа
│                                │
│  ┌──────────────────────────┐  │
│  │                          │  │
│  │    ІГРОВА ЗОНА           │  │  ← 60% висоти екрану
│  │    (залежить від         │  │     border-radius: xl
│  │     вікової групи)       │  │     фон: surface
│  │                          │  │     shadow: card
│  │                          │  │
│  └──────────────────────────┘  │
│                                │
│   Крок 1 з 4     ● ○ ○ ○      │  ← progress dots (кроки туторіалу)
└────────────────────────────────┘
```

### AC-3: Гра для 🐣 малюків (до 4 років) — "Тапни на зірочку!"
- **Given** вікова група = malyuki
- **When** туторіал запускається
- **Then** 4 кроки:

**Крок 1: "Тапни на зірочку!"**
- Персонаж каже (озвучка): "Бачиш зірочку? Натисни на неї!"
- В ігровій зоні: одна велика зірка ⭐ (80dp), по центру, пульсуюча анімація (scale 0.9↔1.1, loop)
- Тап на зірку → зірка "вибухає" (scale 0→2, opacity 1→0, 300ms) + sparkle effect
- Звук: "ding!" (приємний, короткий)
- Персонаж: "Молодець! 🎉" (озвучка)

**Крок 2: "А тепер ще одну!"**
- Персонаж: "А тепер знайди зірочку!" (озвучка)
- Зірка з'являється в випадковому місці ігрової зони (не по центру)
- Тап → такий же ефект
- Персонаж: "Супер!" (озвучка)

**Крок 3: "Ще дві!"**
- Персонаж: "Тепер натисни на обидві зірочки!" (озвучка)
- Дві зірки в різних місцях (не overlapping, мін. 100dp відстань)
- Тап на першу → ефект. Тап на другу → ефект
- Персонаж: "Ти вже все вмієш!" (озвучка)

**Крок 4: Результат**
- Перехід до AC-6 (екран результату)

### AC-4: Гра для 🐼🦊🦉 (5+ років) — "Перетягни в кошик!"
- **Given** вікова група = doskilnyata | pershoklas | drugoklas
- **When** туторіал запускається
- **Then** 4 кроки:

**Крок 1: "Перетягни яблуко в кошик!"**
- Персонаж каже: "Бачиш яблуко? Перетягни його в кошик!" (текст + озвучка)
- Ігрова зона:
  ```
  ┌────────────────────────┐
  │                        │
  │  🍎 (64dp)             │  ← draggable, зліва-зверху
  │                        │
  │                        │
  │            🧺 (80dp)   │  ← drop zone, справа-знизу
  │         ·····          │     dashed border коли drag active
  │         · кошик ·      │     glow effect коли item над зоною
  │         ·····          │
  └────────────────────────┘
  ```
- Drag: `react-native-gesture-handler` PanGesture
- При drag: яблуко слідує за пальцем, scale 1.1, shadow збільшується
- Drop в зону: яблуко зникає (scale→0), кошик "підстрибує" (translateY -10, spring)
- Звук: "pop" при підборі, "success" при drop
- Персонаж: "Молодець!" (озвучка)

**Крок 2: "А тепер банан!"**
- Персонаж: "Тепер перетягни банан!" (озвучка)
- 🍌 зліва → 🧺 справа
- Аналогічно

**Крок 3: "Два фрукти одразу!"**
- Персонаж: "Тепер два фрукти!" (озвучка)
- 🍎 і 🍊 → 🧺
- Порядок не важливий

**Крок 4: Результат**
- Перехід до AC-6

### AC-5: Підказки під час туторіалу
- **Given** будь-який крок туторіалу
- **When** дитина не виконує дію протягом 5 секунд
- **Then** підказка:
  - Персонаж повторює інструкцію (озвучка)
  - Пульсуюча стрілка/finger icon вказує на ціль (animated pointer)
  - Для drag: показується ghost path (пунктирна лінія від item до зони)
- **When** ще 5 секунд бездіяльності
- **Then** повторна підказка з більш явним указівником

### AC-6: Екран результату
- **Given** всі кроки туторіалу виконані
- **When** бачу результат
- **Then** layout:

```
┌────────────────────────────────┐
│                                │
│     [Персонаж аплодує]         │  ← 100dp, анімація celebrate
│                                │
│        ⭐ ⭐ ⭐                 │  ← 3 зірки, з'являються послідовно
│                                │     кожна: scale 0→1.2→1.0, 200ms інтервал
│                                │     золотий колір, sparkle effect
│                                │
│   Молодець, Максимко!          │  ← h1, center
│   Ти вже все вмієш! 🎉        │  ← body, center
│                                │
│       +5 XP                    │  ← h2, secondary color, bounce animation
│                                │
│  ┌──────────────────────────┐  │
│  │    Почнемо пригоду! →    │  │  ← AppButton primary, fullWidth
│  └──────────────────────────┘  │
│                                │
└────────────────────────────────┘
```

- **And** звук: fanfare (той самий що і при виборі аватара, або інший cheerful)
- **And** confetti (легше ніж при аватарі — 10-15 елементів)
- **And** +5 XP додається до прогресу дитини через progressStore
- **And** натискання "Почнемо пригоду!" → navigate до Hub `/(main)/`

### AC-7: Кнопка "Пропустити"
- **Given** екран туторіалу
- **When** натискаю "Пропустити" (верхній правий кут)
- **Then** confirmation (для батьків):
  - Текст: "Пропустити навчання?"
  - Кнопки: "Продовжити навчання" (primary) | "Пропустити" (ghost)
- **When** підтверджую пропуск
- **Then** перехід на Hub без нагороди за туторіал (0 XP)
- **And** "Пропустити": font-size 14sp, textSecondary — непомітна для дитини, видна батькам

### AC-8: Тривалість та перехід між кроками
- **Given** туторіал
- **When** дитина грає
- **Then** загальна тривалість ≤ 1 хвилина (4 кроки по ~10-15 секунд)
- **And** перехід між кроками: fade crossfade (300ms)
- **And** progress dots оновлюються при переході
- **And** персонаж говорить нову фразу на кожному кроці

## Технічні нотатки
- Туторіал hardcoded, НЕ процедурний (на відміну від ігор в EP-05+)
- State machine для кроків: `step: 1|2|3|4`, `status: 'instruction'|'playing'|'complete'`
- Drag-and-drop: `Gesture.Pan()` + `useAnimatedStyle` для position
- Drop zone detection: check if item center is within drop zone bounds (simple rect check)
- Компоненти TapStarGame і DragFruitGame — прототипи для майбутніх ігор (practice для developer)
- XP за туторіал = 5 (фіксовано), зараховується до progressStore
- Озвучка файли: `assets/voice/{lang}/tutorial_{step}.mp3`
