# US-006 — Вибір мови

## User Story
**Як** батько/мати,
**я хочу** обрати мову додатку при першому запуску,
**щоб** дитина бачила контент рідною мовою.

## Acceptance Criteria

### AC-1: Коли показується екран
- **Given** авторизований користувач
- **When** немає жодного профілю дитини (перший вхід після реєстрації)
- **Then** redirect на `/(main)/onboarding/language` — перший крок онбордингу
- **And** back button відсутній (не можна пропустити)
- **And** якщо мова вже обрана (є в settings store) — пропускаємо цей крок

### AC-3: Взаємодія з кнопками мови
- **Given** дві кнопки мови
- **When** тапаю на одну
- **Then** flow:
  1. Обрана кнопка отримує: border 2px `primary`, фон `primaryLight`, scale 1.02 (withSpring, 200ms)
  2. Інша кнопка: opacity 0.5 (withTiming, 200ms)
  3. Через 300ms після тапу — автоматичний перехід на наступний крок
     (НЕ потрібна окрема кнопка "Далі" — тап = вибір + перехід)
  4. Мова зберігається: `settingsStore.setLanguage('uk' | 'en')`
  5. Всі тексти в додатку миттєво перемикаються
  6. Navigate до `/(main)/onboarding/name` з анімацією slide-left (300ms)

### AC-5: Зміна мови пізніше
- **Given** мову обрано при онбордингу
- **When** батько заходить у `/(parent)/settings`
- **Then** є секція "Мова" з поточною мовою та прапором
- **When** тапає → dropdown або bottom sheet з двома опціями (🇺🇦/🇬🇧)
- **Then** мова змінюється миттєво без перезапуску додатку
- **And** зміна зберігається в MMKV через settingsStore

### AC-6: Мова впливає на озвучку
- **Given** мова обрана
- **When** додаток програє озвучку
- **Then** аудіо файли вибираються з відповідної мовної директорії:
  - UK: `assets/voice/uk/koko_greeting.mp3`
  - EN: `assets/voice/en/koko_greeting.mp3`
- **And** якщо аудіо для обраної мови відсутнє → fallback на українську

## Технічні нотатки
- Не використовувати i18next — overkill для 2 мов. Простий lookup:
  ```typescript
  const t = (key: string, params?: Record<string, string>) => {
    const value = key.split('.').reduce((obj, k) => obj?.[k], translations[language]);
    if (!value) return key; // fallback
    return params ? value.replace(/\{\{(\w+)\}\}/g, (_, k) => params[k] ?? '') : value;
  };
  ```
- JSON ключі: nested для організації, dot notation для доступу
- `useTranslation` hook: бере language з settingsStore (Zustand), підписується на зміни
- При зміні мови — всі підписані компоненти автоматично ре-рендеряться (Zustand reactivity)
