# US-008 — Вибір аватара

## User Story
**Як** дитина (з допомогою батьків),
**я хочу** обрати собі аватар,
**щоб** мій профіль був персональним і цікавим.

## Acceptance Criteria

### AC-1: Layout екрану вибору аватара
- **Given** профіль дитини створено (ім'я + вік)
- **When** бачу екран `/(main)/onboarding/avatar`
- **Then** layout:

```
┌────────────────────────────────┐
│  ← (back → age-group)          │
│                                │
│  Обери свого героя,            │  ← AppText h1, center
│  Максимко! 🎭                  │  ← ім'я з профілю
│                                │
│  ┌────┐  ┌────┐  ┌────┐       │
│  │ 🐱 │  │ 🐶 │  │ 🐰 │       │  ← сітка 3x2 (або 2x3 на portrait)
│  │    │  │    │  │    │       │     кожен аватар: 80x80dp
│  └────┘  └────┘  └────┘       │     відступ між: 16dp
│  ┌────┐  ┌────┐  ┌────┐       │     border-radius: full (круглий)
│  │ 🦋 │  │ 🐻 │  │ 🦄 │       │     фон: surface з тінню card
│  │    │  │    │  │    │       │
│  └────┘  └────┘  └────┘       │
│                                │
│  ┌──────────────────────────┐  │
│  │       Це я! ✨            │  │  ← AppButton primary, fullWidth
│  └──────────────────────────┘  │     disabled поки не обрано
│                                │
│   Крок 3 з 3    ● ● ●         │  ← step indicator
└────────────────────────────────┘
```

- **And** 6 стартових аватарів — яскраві тваринки або персонажі
- **And** на tablet/web: max width 400dp, центровано
- **And** кнопка "Це я!" disabled поки жоден аватар не обрано

### AC-2: 6 стартових аватарів
- **Given** сітка аватарів
- **When** дивлюсь на варіанти
- **Then** 6 аватарів:

| # | Аватар | Emoji placeholder | Стиль |
|---|--------|------------------|-------|
| 1 | Котик | 🐱 | Помаранчевий котик з бантиком |
| 2 | Собачка | 🐶 | Золотистий ретрівер |
| 3 | Зайчик | 🐰 | Білий з рожевими вушками |
| 4 | Метелик | 🦋 | Фіолетовий з блискітками |
| 5 | Ведмедик | 🐻 | Коричневий з шарфиком |
| 6 | Єдиноріг | 🦄 | Рожево-фіолетовий |

- **And** MVP: emoji 40sp на кольоровому circle (80x80dp)
- **And** Phase 2: заміна на PNG/SVG ілюстрації 256x256
- **And** кольори фону аватарів (circle background):
  1. `#FFE0B2` (теплий помаранчевий)
  2. `#FFF9C4` (світло-жовтий)
  3. `#F8BBD0` (рожевий)
  4. `#E1BEE7` (лавандовий)
  5. `#D7CCC8` (бежевий)
  6. `#F3E5F5` (світло-фіолетовий)

### AC-3: Анімація вибору
- **Given** сітка аватарів, жоден не обрано
- **When** тапаю на аватар
- **Then** анімація:
  1. Обраний: scale 1.15 (withSpring, stiffness: 200, damping: 10) — "bounce up"
  2. Обраний: ring border 3px `primary` з'являється (withTiming, 200ms)
  3. Обраний: тінь збільшується (elevation: 6)
  4. Інші 5: scale 0.9, opacity 0.4 (withTiming, 200ms)
  5. Звуковий ефект: "pop" (100ms)
  6. Кнопка "Це я!" стає активною (opacity 0.5 → 1.0, withTiming 200ms)

- **When** тапаю на інший аватар
- **Then** попередній повертається до default, новий анімується (як вище)

### AC-4: Підтвердження та збереження
- **Given** аватар обрано
- **When** натискаю "Це я! ✨"
- **Then** flow:
  1. Аватар зберігається в профілі: `childProfilesStore.updateProfile(id, { avatar: 'cat' })`
  2. MMKV persist
  3. Показується вітальна анімація (AC-5)
  4. Після анімації → перехід до туторіалу (US-009) або Hub

### AC-5: Вітальна анімація
- **Given** аватар підтверджено
- **When** показується вітальна анімація
- **Then** overlay на весь екран (2.5 секунди):

```
┌────────────────────────────────┐
│                                │
│                                │
│     [Обраний аватар]           │  ← scale animation: 0 → 1.2 → 1.0 (bounce)
│        80dp → 120dp            │
│                                │
│   ✨ Чудово, Максимко! ✨      │  ← AppText h1, fadeIn після 0.5s
│                                │
│   [Персонаж-компаньйон]       │  ← маленький (40dp), знизу
│   аплодує 👏                   │  ← анімація "celebrate"
│                                │
│                                │
└────────────────────────────────┘
```

- **And** фон: `theme.background` з confetti анімацією (прості кольорові кружечки falling)
- **And** confetti: 20-30 елементів, різних кольорів (з palette), falling + rotation
- **And** звук: fanfare/celebration (1.5s)
- **And** анімація автоматично завершується через 2.5s → navigate далі
- **And** тап на екран → пропустити анімацію (skip)

### AC-6: Зміна аватара пізніше
- **Given** аватар обрано при онбордингу
- **When** натискаю "Аватар" на Hub або через parent panel
- **Then** відкривається екран зміни аватара:
  - Показані всі доступні: 6 стартових + розблоковані (Phase 3)
  - Поточний аватар має border ring
  - Тап на інший → зміна без confirmation animation
  - Зміна зберігається одразу (без кнопки "Зберегти")
  - Navigate back до попереднього екрану

### AC-7: Аватар відображається скрізь
- **Given** аватар обрано
- **When** він показується в UI
- **Then** відображається:
  - Hub: поруч з іменем (32dp circle)
  - Parent panel: у списку профілів (48dp circle)
  - Game result: поруч з результатом (40dp circle)
- **And** якщо аватар = null (ще не обрано) → показується default gray circle з "?"

## Технічні нотатки
- Аватари MVP: emoji на кольоровому circle (простий View з borderRadius: 9999)
- Confetti: можна реалізувати через Reanimated (20 Animated.View з random position, opacity, rotation)
- Або використати `react-native-confetti-cannon` (lightweight, ~5KB)
- `AvatarCircle` component: приймає `avatarId` або `emoji` + `size` → рендерить circle
- Phase 3 (US-056): кастомізація аватара (шапки, окуляри) — тоді замість emoji буде composite image
